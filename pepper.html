Getting Started with Pepper
 

With this tutorial you will learn:

  • How to install Pepper software in your PC (Windows)
  • How to connect Pepper to RIZE

Install Software of Pepper (Windows)
 

Pepper require the installation of Python 2.7 from the official Python website

 

We highly recommend to install Python 2.7 in Windows following the next steps (click in each step to see details, steps 2 is very important):

Download Python 2.7.16 x86 installer from the next link: mdi-download Download Python 2.7.16 x86
 
 Execute the Python 2.7.16 x86 installer with the next configurations (IMPORTANT: click here for details) VERY IMPORTANT when reaching the interface shown below, be sure of activating Add python.exe to Path - Will be installed on local hard drive. This is an error-prone step, be careful.
 
 
 Continue and finish the installation of Python 2.7
 
 
 

Download and install NAO SDK from the next link:

mdi-download Download NAO SDK

Connect Pepper to RIZE

 
Pepper and your PC must be in the same Wifi network. A very common error is to have Pepper and the PC in different Wifi networks.
 

Open RIZE for Pepper robot and select the button mdi-ip to set the IP address of your robot

 
 

Press the Pepper's button in its chest and quickly release it (less than a second) to hear the IP address of the robot.

 
 

Then, write the IP address that the robot dictates in the window opened in RIZE with the button mdi-ip. After writing the right IP address press the button CHANGE IP to set the value

 
 

In the home page press the button CHANGE IP to connect Pepper to RIZE
